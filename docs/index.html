How It Works

Two simple check-in methods designed for speed and convenience.

ID Scan ~5 seconds

Customer scans the barcode on the back of their driver's license.

  • 1
    Customer taps "Scan ID" button on kiosk
  • 2
    Scans back of driver's license (PDF417 barcode)
  • 3
    System extracts: Name, DOB, Address, City, State, ZIP, Gender
  • 4
    Searches local database for existing customer
  • 5
    Prompts for loyalty signup (if not a member)
  • 6
    Added to POSaBIT queue instantly

Guest Entry ~15 seconds

Manual name entry for customers without ID or who prefer typing.

  • 1
    Customer taps "Enter Name" button on kiosk
  • 2
    Types first and last name on touchscreen keyboard
  • 3
    Prompted: "Join our loyalty program?"
  • 4
    If yes: Can scan ID for easy entry OR type phone/email
  • 5
    Customer record created in POSaBIT (if loyalty)
  • 6
    Added to queue as member or guest

Installation Process

Get up and running in under 5 minutes per location.

1. Download Installer

Go to GitHub releases page and download the latest .exe installer. One-click download.

2. Run Setup

Double-click installer. One-click install with no configuration needed. Creates desktop shortcut.

3. Select Venue

On first launch, select your store location from dropdown. This configures the POSaBIT connection.

4. Initial Sync

App syncs customer database in background. Takes 1-2 minutes. App is usable immediately.

5. Connect Scanner

Plug in USB barcode scanner. No drivers needed - works as keyboard input automatically.

6. Ready!

Kiosk launches in fullscreen mode. Ready to check in customers. Auto-updates enabled.
